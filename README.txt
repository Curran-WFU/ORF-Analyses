This repository accompanies the BioRxiv manuscript, Analyses of the 5’ Ends of Escherichia coli ORFs by Michael Ward and James F. Curran.  Doi: 


There are two files:  


“E.coli.K12.sequence.2.gb”  contains the genomic data.  


“ORF.Analyses.0.ipynb” is an analytical program that will reproduce the plots in the manuscript.  In the routines for figures 3 & 4, one may need to change the data structures used for the plotting routine.
